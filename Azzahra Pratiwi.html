Keseharian Saya

Nama saya Azzahra Pratiwi, saya kelas XI RPL 1. Saya ingin bercerita tentang aktivitas saya sehari-hari, dari saya bengun tidur di pagi hari lalu kembali tidur di malam hari. Saya bangun dari tidur saya pada pukul 04.40 WIB. Lalu pada pukul 04.50 WIB saya mandi, setelah itu saya sholat shubuh, habis sholat shubuh saya sarapan pagi. Setelah selesai sarapan saya bersiap untuk berangkat ke sekolah seperti merapikan buku dan merapikan bekal. Dan setelah itu saya berangkat ke sekolah pada pukul 06.15 WIB menggunakan transportasi umum. Lalu saya sampai di sekolah pada pukul 06.45 WIB.

Saya memulai pelajaran pada pukul 07.30 WIB. pada pukul 09.30 WIB, bell istirahat berbunyi lalu saya bersama teman-teman saya pergi ke kantin setelah itu memakan bekal di kelas. Istirahat selesai pada jam 10 lalu saya kembali memulai pelajaran di kelas. Saya belajar di sekolah sampai jam 12 siang, bell pulang berbunyi tepat pada jam 12 siang, lalu saya bersama teman saya pulang ke rumah menggunakan transportasi umum. Saya sampai dirumah pada pukul 12.40 WIB, lalu saya mengganti pakaian seragam saya dengan pakaian keseharian saya. Setelah itu, saya menunaikan sholat Dzuhur. Setelah selesai sholat saya makan siang dan membantu mama saya memasak. Lalu pada pukul 13.40 WIB, saya istirahat dengan tidur siang sampai pukul 15.20 WIB.

Setelah bangun tidur, saya mandi dan setelah itu saya menunaikan sholat Ashar. Lalu pada pukul 16.30 WIB, saya mengajak adik perempuan saya bermain di halaman rumah bersama anak-anak tetangga sekitar. Pukul 17.40 WIB, saya masuk ke rumah karena hari sudah mulai memasuki Maghrib. Setelah itu pada pukul 18.10 WIB, saya menunaikan sholat Maghrib sampai memasuki waktu menunaikan sholat Isya. Setelah selesai beribadah, saya belajar pada pukul 19.30 WIB sampai pukul 20.30 WIB. Dan pada Pukul 21.20 WIB saya terlelap tidur. Sekian keseharian saya sebagai pelajar, menyenangkan tapi terkadang sedikit melelahkan, Terimakasih.
